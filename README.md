**Specifications :**
- **Language of Back-End Development :** Python 3.6
- **Web Framework :** Flask 1.0.2
- **Front-End :** HTML, CSS, Javascript
- **Database :** MongoDB (mLab.com)
- **Information Retrieval Tool :** Beautiful Soup 4.4.0
